# Moodle Installer

## Introduction

This script is for installing Moodle LMS on LAPP stack (Linux, Apache, PostreSQL, PHP) in CentOS 8
___

## Installed Packages

This script will install the followings:

|  Packages  | Version |  Remarks |
|:----------:|:-------:|:--------:|
| Apache     | 2.4.46  | httpd    |
| PostgreSQL | 10.6    | psql     |
| PHP        | 7.2     | php      |
